Виконати наступне:
1. Створити локальний репозиторій (консоль)
2. Створити новий пустий репозиторій на GitHub (інтерфейс GitHub)
3. Приєднати свій локальний репозиторій до новоствореного на GitHub (консоль)
4. Створити новий файл ‘commands.txt’ із будь-яким контентом (редактор коду)
5. Зробити commit доданого файлу (консоль)
6. Зробити push доданого файлу на віддалений репозиторій на GitHub (консоль)
7. Створити нову гілку new-branch (консоль)
8. Перейти на нову гілку (консоль)
9. Добавити стрічку ‘new branch changes’ в кінець файлу ‘‘commands.txt’ (редактор коду)
10. Зробити commit файлу (консоль)
11. Зробити push файлу з локальної гілки new-branch на віддалену гілку new-branch (віддалена гілка автоматично створиться, якщо її не буде) (консоль)
12. Переключитись назад на гілку master (консоль)
13. Змерджити гілку new-branch на гілку master (поточна) (консоль)
14. Зробити push гілки master у гілку master у віддаленому репозиторії на GitHub (консоль)
15. Видалити з файла commands.txt весь контент і добавити список команд, за допомогою яких ви робили завдання (для кожного пункту завдання, який потрібно зробити за допомогою консолі).
16. Закомітити зміни (консоль).
17. Зробити push у віддалений репозиторій (консоль). (edited)
